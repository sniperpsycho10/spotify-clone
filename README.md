# Spotify Clone

A full-stack Spotify-inspired music streaming application built using React, Node.js, and MongoDB.

## Features (Planned)

* User Authentication
* Music Playback
* Song Upload
* Playlist Creation
* Search Functionality
* Recently Played Songs
* Responsive Design
* Online Deployment

## Tech Stack

### Frontend

* React
* Vite
* React Router
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB
